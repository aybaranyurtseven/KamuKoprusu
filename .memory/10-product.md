# Ürün Tanımı

## Problem Tanımları
1. Vatandaşların kamu kurumlarına şikayetlerini iletmekte zorlanması
2. Başvuru durumlarının takip edilememesi
3. Kurumların vatandaş geri bildirimlerini organize edememesi
4. Şeffaflık eksikliği

## Kullanıcı Profilleri

### Vatandaş
- Şikayet/öneri oluşturur
- Medya dosyası ekler
- Başvuru durumunu takip eder
- Rozet ve seviye kazanır

### Kurum Temsilcisi
- Gelen başvuruları inceler
- Durum günceller
- Geri bildirim yazar
- Çözüm kanıtı yükler

### STÖ Koordinatörü
- Vatandaşları destekler
- Ekip yönetir

### Moderatör
- İçerik inceler
- Uyarı verir
- Kullanıcı yasaklar

### Admin
- Tüm sistemi yönetir
- İstatistik izler
- Kurum onaylar

## Temel Özellikler

### Vatandaş Sayfaları
| Sayfa | Açıklama |
|-------|----------|
| CreateComplaint | Şikayet/öneri oluşturma |
| MyComplaints | Başvuru listesi |
| ComplaintDetail | Detay görüntüleme |
| Dashboard | İstatistikler |

| Achievements | Rozetler |

| Sayfa | Açıklama |

| Sayfa | Açıklama |

- Çözülen başvuru: 25 puan
- Medya ekleme: +5 puan
